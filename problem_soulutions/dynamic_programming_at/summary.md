# AtCoder Educational DP Contest

Welcome to the AtCoder Educational DP Contest section! This comprehensive collection covers 26 dynamic programming problems designed to teach and reinforce various DP techniques from basic to advanced.

## Overview

The AtCoder Educational DP Contest is a curated set of problems that progressively build your understanding of dynamic programming. Each problem is labeled from A to Z and covers different DP patterns and techniques.

## Problem Categories

### Basic DP (Problems A-E)

#### **1D Linear DP**
- **[A - Frog 1](frog_1_analysis)** - 1D DP with linear transitions
- **[B - Frog 2](frog_2_analysis)** - 1D DP with k-step transitions
- **[Z - Frog 3](frog_3_analysis)** - Advanced DP with convex hull trick

#### **State Machine DP**
- **[C - Vacation](vacation_analysis)** - State machine DP with constraints

#### **Knapsack Problems**
- **[D - Knapsack 1](knapsack_1_analysis)** - Classic 0/1 knapsack
- **[E - Knapsack 2](knapsack_2_analysis)** - Value-based DP (state inversion)

### Graph & Grid DP (Problems G-H)

- **[G - Longest Path](longest_path_analysis)** - DP on Directed Acyclic Graphs (DAGs)
- **[H - Grid 1](grid_1_analysis)** - 2D grid DP with obstacles

### Probability & Game Theory (Problems I-K)

- **[I - Coins](coins_analysis)** - Probability DP
- **[J - Sushi](sushi_analysis)** - Expectation DP with state compression
- **[K - Stones](stones_analysis)** - Game theory DP

### Advanced DP Patterns (Problems L-M)

- **[L - Deque](deque_analysis)** - Interval DP
- **[M - Candies](candies_analysis)** - DP with prefix sum optimization

### Interval & Bitmask DP (Problems N-O)

- **[N - Slimes](slimes_analysis)** - Interval DP with merging
- **[O - Matching](matching_analysis)** - Bitmask DP

### Tree DP (Problems P-V)

- **[P - Independent Set](independent_set_analysis)** - Tree DP
- **[Q - Flowers](flowers_analysis)** - DP with segment tree
- **[V - Subtree](subtree_analysis)** - Tree DP with rerooting

### Advanced Techniques (Problems R-Z)

- **[R - Walk](walk_analysis)** - Matrix exponentiation
- **[S - Digit Sum](digit_sum_analysis)** - Digit DP
- **[T - Permutation](permutation_analysis)** - Insertion DP
- **[U - Grouping](grouping_analysis)** - Subset DP with bitmask
- **[W - Intervals](intervals_analysis)** - Interval covering DP
- **[X - Tower](tower_analysis)** - Box stacking DP
- **[Y - Grid 2](grid_2_analysis)** - Inclusion-exclusion DP
- **[Z - Frog 3](frog_3_analysis)** - Convex hull trick

## Learning Path

### Beginner Level (A-C)
Start with these fundamental problems:
1. **Frog 1** - Learn basic 1D DP
2. **Frog 2** - Extend to multiple transitions
3. **Vacation** - Understand state machine DP

### Intermediate Level (D-M)
Build on fundamentals:
4. **Knapsack 1 & 2** - Master knapsack variants
5. **Grid 1** - Learn 2D DP
6. **Longest Path** - DP on graphs
7. **Coins, Sushi** - Probability and expectation DP
8. **Deque, Slimes** - Interval DP

### Advanced Level (N-Z)
Challenge yourself with advanced techniques:
9. **Matching** - Bitmask DP
10. **Tree DP problems** - Independent Set, Subtree
11. **Matrix Exponentiation** - Walk
12. **Digit DP** - Digit Sum
13. **Convex Hull Trick** - Frog 3

## Key DP Techniques Covered

## Tips for Success

1. **Start with Basics**: Don't skip the easy problems - they build essential foundations
2. **Understand State**: Clearly define what your DP state represents
3. **Practice Transitions**: Master writing recurrence relations
4. **Optimize Gradually**: First make it work, then optimize
5. **Study Patterns**: Recognize common DP patterns across problems
