# B - Blue Problems

Welcome to the Blue problem set! This collection contains competitive programming problems organized by algorithmic topics across 19 sessions plus a final exam.

| Topic | Description |
|-------|-------------|
| [Dynamic Array](/cses-analyses/problem_soulutions/Blue/Session%2001%20-%20Dynamic%20Array/) | Array manipulation and basic data structures |

| [Stack and Queue](/cses-analyses/problem_soulutions/Blue/Session%2004%20-%20Stack%20and%20Queue/) | Linear data structures |
| [String](/cses-analyses/problem_soulutions/Blue/Session%2005%20-%20String/) | String manipulation and pattern matching |
| [BFS](/cses-analyses/problem_soulutions/Blue/Session%2006%20-%20BFS/) | Breadth-first search traversal |
| [DFS](/cses-analyses/problem_soulutions/Blue/Session%2007%20-%20DFS/) | Depth-first search traversal |
| [Heap](/cses-analyses/problem_soulutions/Blue/Session%2008%20-%20Heap/) | Priority queue and heap operations |
| [Dijkstra](/cses-analyses/problem_soulutions/Blue/Session%2009%20-%20Dijkstra/) | Shortest path algorithm |
| [Mid Term](/cses-analyses/problem_soulutions/Blue/Session%2010%20-%20Mid%20Term/) | Midterm exam problems |

## Topics Covered

- **Data Structures**: Arrays, Stacks, Queues, Heaps, Trees, Tries, DSU
- **Graph Algorithms**: BFS, DFS, Dijkstra, Bellman-Ford, Floyd-Warshall, Prim, Kruskal
- **Searching**: Binary Search, BST operations
- **String Processing**: String manipulation and algorithms
